# brian-musanga-portfolio — Project Rules

## Design Rules

Fonts are locked: Instrument Serif (display), General Sans (body), JetBrains Mono (labels). Never introduce Inter, Poppins, Space Grotesk, or any other font.

Icons: typographic glyphs (↗ →) preferred; @phosphor-icons/react weight=light only when a true icon is required; @icons-pack/react-simple-icons for brand logos only.
